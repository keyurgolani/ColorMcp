# Analyze Color - Use Case 5: Brand Accent Color Assessment

## Scenario

Brand designer evaluating a purple accent color for premium product branding.

## Input Parameters

```json
{
  "color": "#8B5CF6"
}
```

## Output

```json
{
  "data": {
    "analysis": {
      "brightness": {
        "perceived_brightness": 124,
        "brightness_category": "medium"
      },
      "temperature": {
        "temperature": "cool",
        "hue_category": "blue",
        "kelvin_approximation": 6500
      },
      "accessibility": {
        "wcag_aa_normal": true,
        "wcag_aa_large": true,
        "color_blind_safe": true
      }
    },
    "summary": {
      "overall_score": 90,
      "strengths": ["Good contrast", "Color-blind friendly"]
    }
  }
}
```

## Key Insights

- Excellent for premium brand positioning
- Color-blind safe ensures broad accessibility
- Cool temperature conveys sophistication and trust
