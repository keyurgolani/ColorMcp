# Generate Harmony Palette - Use Case 5: Dark Theme Monochromatic Scale

## Scenario

UI designer creating a sophisticated monochromatic dark theme with subtle variations.

## Input Parameters

```json
{
  "base_color": "#0F172A",
  "harmony_type": "monochromatic",
  "count": 7,
  "variation": 40
}
```

## Output Summary

```json
{
  "data": {
    "palette": [
      { "hex": "#0f172a", "brightness": 23 },
      { "hex": "#111622", "brightness": 22 },
      { "hex": "#0f1524", "brightness": 21 },
      { "hex": "#0f1729", "brightness": 23 },
      { "hex": "#111d38", "brightness": 28 },
      { "hex": "#122248", "brightness": 34 },
      { "hex": "#122759", "brightness": 38 }
    ],
    "metadata": {
      "harmonyType": "monochromatic",
      "harmonyScore": 100,
      "diversity": 4
    }
  }
}
```

## Key Insights

- Perfect harmony score (100) for cohesive dark theme
- Subtle brightness variations (21-38) for depth
- All colors maintain excellent accessibility
- Ideal for sophisticated dark UI designs
