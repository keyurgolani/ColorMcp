# Generate Harmony Palette - Use Case 2: Nature-Inspired Analogous Palette

## Scenario

Environmental website designer creating a calming, nature-inspired color scheme.

## Input Parameters

```json
{
  "base_color": "#10B981",
  "harmony_type": "analogous",
  "count": 6,
  "variation": 15
}
```

## Output Summary

```json
{
  "data": {
    "palette": [
      { "hex": "#10b981", "temperature": "cool" },
      { "hex": "#0fba46", "temperature": "neutral" },
      { "hex": "#0fb56b", "temperature": "cool" },
      { "hex": "#11b28f", "temperature": "cool" },
      { "hex": "#0fafb1", "temperature": "cool" },
      { "hex": "#108fb7", "temperature": "cool" }
    ],
    "metadata": {
      "harmonyType": "analogous",
      "harmonyScore": 84,
      "diversity": 8
    }
  }
}
```

## Key Insights

- Smooth green-to-blue transition
- High harmony score (84) creates cohesive feel
- Perfect for nature/environmental themes
- All colors maintain excellent accessibility
