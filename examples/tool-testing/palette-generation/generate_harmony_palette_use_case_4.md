# Generate Harmony Palette - Use Case 4: Creative Tetradic Art Project

## Scenario

Digital artist creating a vibrant tetradic palette for abstract artwork.

## Input Parameters

```json
{
  "base_color": "#8B5CF6",
  "harmony_type": "tetradic",
  "count": 4,
  "variation": 30
}
```

## Output Summary

```json
{
  "data": {
    "palette": [
      { "hex": "#8b5cf6", "temperature": "cool" },
      { "hex": "#f65a79", "temperature": "warm" },
      { "hex": "#c8f65a", "temperature": "warm" },
      { "hex": "#5af6d7", "temperature": "cool" }
    ],
    "metadata": {
      "harmonyType": "tetradic",
      "harmonyScore": 100,
      "diversity": 40
    },
    "relationships": [
      { "relationship": "tetradic", "angle": 90 },
      { "relationship": "complementary", "angle": 180 }
    ]
  }
}
```

## Key Insights

- Perfect 90° spacing creates balanced composition
- Two complementary pairs for maximum contrast
- High diversity (40) ensures visual interest
- Excellent for creative and artistic applications
