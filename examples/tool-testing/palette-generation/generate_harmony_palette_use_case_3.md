# Generate Harmony Palette - Use Case 3: Bold Triadic Data Visualization

## Scenario

Data visualization designer creating a vibrant triadic palette for dashboard charts.

## Input Parameters

```json
{
  "base_color": "#E11D48",
  "harmony_type": "triadic",
  "count": 3,
  "variation": 25
}
```

## Output Summary

```json
{
  "data": {
    "palette": [
      { "hex": "#e11d48", "temperature": "warm" },
      { "hex": "#48e21d", "temperature": "neutral" },
      { "hex": "#1d48e2", "temperature": "cool" }
    ],
    "metadata": {
      "harmonyType": "triadic",
      "harmonyScore": 100,
      "diversity": 40
    },
    "relationships": [{ "relationship": "triadic", "angle": 120 }]
  }
}
```

## Key Insights

- Perfect 120° spacing creates maximum visual impact
- High diversity (40) ensures clear data distinction
- Perfect harmony score (100) for balanced design
- Red-green-blue primary triadic relationship
